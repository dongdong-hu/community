git 操作步骤:
   1 git init 仓库初始化
   2 git status 查看git状态
   3 git add . 把文件放到暂存区
   4 git remote add origin https://github.com/dongdong-hu/community.git  第一次提交的时候 需要指向需要提到哪个仓库里面去
   4.1 git push -u origin master 推到远程分支
   5 第一次 提交成功
   6 第二次 提交
   7 git add .
   8 git commit -m "备注"
   9 git push git 
   10 追加 提交  git commit --amend --no-edit
